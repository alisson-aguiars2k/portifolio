minha jornada

Minha Jornada começa em 2021, com as playlists; HTML e CSS3 módulos 1 e 2, JavaScript e ECMAScript para iniciantes, no canal Curso em Video do querido professor Gustavo Guanabara.

Na playlist, pratiquei bastante a linguagem de marcação de forma semântica e a linguagem de estilização com conceitos de Tipografia das Fontes e Pisicologia das Cores Desenvolvendo assim projetos, colocando em prática os conhecimentos adquirido ao longo dos módulos. No JavaScript e ECMAScript para iniciantes, aprendi conceitos básicos sobre Variáveis, Operadores e a Manipulação de DOM.

No final de 2021, estava convicto que essa era a área que gostaria de se profissionalizar, além de, no futuro próximo, fazer especializações, mestrado e doutorado.

Analizando o mercado atual, senti a nescessidade de comprar cursos mais avançados com projetos voltado para o "mundo real" na intenção de continuar evoluindo e se aprofundando nas tecnologias voltadas para o Desenvolvimento Web ( Front-End ).

Após meses de muitas buscas, assinei com a plataforma Rocketseat ( discover ) , B7web ( fullstack ) e CodingAddict, com foco na prática, em uma formação imersiva em tecnologia.
Nessas plataformas, potencializei meus conhecimentos com também aprendi novas tecnologias ;
☑️ HTML semântico;
☑️ CSS3 com Flexbox, Grid Layout com Responsive.
☑️ JavaScript aprendendo sobre Variáveis, Manipulação do DOM, Consultas de APIs, Requisições Síncrona e Assícrona. Adicionando assim interatividades as aplicações.
☑️ TypeScript em Arrays, Objects e Functions, e tsConfig.
☑️ Na biblioteca React-js aprendedo as States e Events, Requisições com padrões RESTful GET, POST, PUT e DELETE, os Hooks useState, useEffect, useReducer, useRef e useContext, rotas com React Router 6, Redux, Axios HTTP Library. Sempre procurando confrontar a teoria com a prática, fazendo projetos e desafios durante as aulas.
☑️ Framework CSS inseridos nos desafios e projetos ; Styled-Components e Tailwind.
☑️ Testes Unit e Integration com Jest-js e Testing Library.
☑️ SQL com ( mySQL, postgreSQL ).
☑️ NoSQL com ( MangoDB ).
☑️ Git fazendo Repositório Local( Verificar estado, Adicionar arquivo, Comitar arquivo, Remover arquivo, Visualizar histórico, Desfazendo operações).
☑️ Github fazendo Repositório Remoto( Exibir os repositórios remotos, Vincular repositório local com um repositório remoto, Exibir informações dos repositórios remotos, Desvincular um repositório remoto, Visualizar histórico, Enviar arquivos/diretórios para o repositório remoto, Atualizar repositório local de acordo com o repositório remoto, Clonar um repositório remoto já existente, Branches, Rebasing).

☑️ Nocões básicas Scrum( Sprints de trabalho curtos e estruturados ).


Atualmente estou finalizando os estudos nas plataformas e planejando se especializar na trilha ( React-js ) da Rocketseat Ignite para continuar evoluindo como programador e buscar as melhores opotunidades no mercado de trabalho.
